Welcome to NutriSphere's Admin Portal

NutriSphere is building a database-driven calorie-tracking system to help health coaches manage client nutrition and exercise data more efficiently.

The system replaces manual tracking with a structured approach, integrating Users, Exercises, DailyTrackers, FoodItems, and FoodEntries to store profiles, calorie goals, exercise logs, and food intake.

Initially supporting 300 users, the platform provides actionable insights into nutrition and fitness, streamlining data management and improving user experience.

By transitioning to a digital solution, NutriSphere enhances efficiency, accuracy, and long-term health tracking.

Database Overview

NutriSphere's database consists of key tables to track users, food, and exercise data efficiently.

Table Description
Users Stores user profiles including username, email, and daily calorie goals.
Daily Trackers Tracks daily calorie intake and exercise records for each user.
Food Entries Logs foods consumed by users, linked to daily trackers.
Food Items Contains nutritional data for various food items, including calories, protein, fat, and carbohydrates.
Exercises Lists exercise types, minutes performed, and calories burned.

Reset All Data

Resets Users, Exercises, Food Items tables with default data, and fully clear Daily Trackers and Food Entries (empty state).
Warning: This action is irreversible.
